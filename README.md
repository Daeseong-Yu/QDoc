# QDoc

QDoc is a patient-facing queue and check-in experience for walk-in care.

QDoc lets a user sign in, search nearby hospitals, join a queue, and monitor live queue status from the web app.

## What QDoc Does

- Search nearby hospitals and compare queue conditions
- Check in to a hospital queue
- View live queue status with WebSocket updates and polling fallback
- Send a notification when the queue is about to reach the customer.
